--------------
Function
--------------
.mod assigner creates a .mod file for every folder in "mods" directory with the contents
from the included descriptor.mod and proceeds to create a path to the mod's folder
--------------
Usage
--------------
1. Put mod folders in "mods" directory
2. Run m_assi.py
3. Done!

NOTE: You'll probably have to edit the m_assi.py file to add a correct path
